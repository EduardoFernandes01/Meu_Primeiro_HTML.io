Formulários

Formulários em HTML são estruturas que permitem a interação entre o usuário e uma aplicação ou site. Eles servem como uma interface de coleta de dados, onde o visitante pode inserir informações que serão enviadas e processadas por um servidor ou por scripts no próprio navegador.

Form e seus atributos

Um formulário é definido pela tag <form> e dentro dela ficam os elementos de entrada (inputs), como campos de texto, caixas de seleção, botões, áreas de texto, listas suspensas, entre outros. Mais a frente serão mostrados alguns exemplos, agora vamos focar em alguns atributos que essa tag carrega, sendo alguns deles:

   
    method: Define o método HTTP a ser usado ao enviar o formulário. Os valores comuns são "GET" e "POST".

    action: Especifica o URL para onde os dados do formulário serão enviados quando o formulário for submetido.

    name: Atribui um nome ao formulário, que pode ser usado para referenciá-lo em scripts.

    target: Define onde abrir a página de resposta após o envio do formulário. Valores comuns incluem "_self" (mesma aba), "_blank" (nova aba), "_parent" e "_top".

    autocomplete: Indica se o navegador deve ou não preencher automaticamente os campos do formulário com base em entradas anteriores. Valores possíveis são "on" (ativado) e "off" (desativado).

    onsubmit: Especifica um script a ser executado quando o formulário é submetido. Pode ser usado para validação de dados antes do envio.

    onreset: Especifica um script a ser executado quando o formulário é resetado.

Em suma, os formulários são essenciais para coletar informações dos usuários em páginas web. Eles permitem a criação de interfaces interativas e personalizadas, facilitando o envio de dados para processamento, seja por servidores ou scripts locais. O domínio dos atributos e elementos de um formulário é fundamental para o desenvolvimento de aplicações web eficientes e seguras.

Labels

Labels são elementos HTML representados pela tag <label>, usados para associar textos descritivos a campos de formulário, como inputs, selects ou textareas. Eles melhoram a acessibilidade e a usabilidade, pois ao clicar em um label, o campo correspondente é ativado.

Inputs e seus tipos

Inputs são elementos HTML representados pela tag <input>, usados para criar campos interativos em formulários. Eles permitem ao usuário inserir dados, como texto, números, senhas, datas, arquivos e muito mais. São usados em conjunto com as labels para dar clareza no texto que deve ser inserido dentro do campo de texto. Abaixo temos alguns exemplos de como usar os inputs juntamente com as labels:

  • text: Campo de texto simples.
    • Código: <label>Nome:</label><input type="text" name="nome">


  • password: Campo para senha, oculta os caracteres digitados.
    • Código: <label>Senha:</label><input type="password" name="senha">


  • email: Campo para e-mail, valida o formato.
    • Código: <label>E-mail:</label><input type="email" name="email">


  • number: Campo para números.
    • Código: <label>Idade:</label><input type="number" name="idade">


  • date: Campo para seleção de data.
    • Código: <label>Data de Nascimento:</label><input type="date" name="nascimento">


  • week: Campo para seleção de uma semana do ano.
    • Código: <label>Semana:</label><input type="week" name="semana">


  • month: Campo para seleção de um mês específico.
    • Código: <label>Mês:</label><input type="month" name="mes">


  • checkbox: Caixa de seleção (marcar/desmarcar).
    • Código:
<label>Adicionais:</label>
<input type="checkbox" name="adicionais[]" value="queijo"> + Queijo
<input type="checkbox" name="adicionais[]" value="bacon"> + Bacon
<input type="checkbox" name="adicionais[]" value="tomate"> + Tomate
<input type="checkbox" name="adicionais[]" value="picles"> + Picles
    + Queijo + Bacon + Tomate + Picles

    Obs.: Note que no checkbox podemos selecionar múltiplos valores ao mesmo tempo. Quando o formulário for enviado, será criado um vetor chamado 'adicionais', com os valores que foram marcados inseridos dentro dele.



  • radio: Botão de opção, permite selecionar apenas uma opção de um grupo.
    • Código:
<label>Sexo:</label>
<input type="radio" name="sexo" value="M"> Masculino
<input type="radio" name="sexo" value="F"> Feminino
    Masculino Feminino

  • range: Cria uma barra com valores mínimo, máximo e valor inicial definidos.
    • Código: <label>Volume:</label><input type="range" name="volume" min="0" max="100" value="50">


  • color: Cria um campo seletor de cores.
    • Código: <label>Cor favorita:</label><input type="color" name="cor">


  • url: Cria um campo de formulário específico para inserção de endereços de sites.
    • Código: <label>Site:</label><input type="url" name="site">


  • file: Permite o envio de 1 ou mais arquivos.
    • Código: <label>Arquivo:</label> <input type="file" name="arquivo" >

    Obs.: Para anexar mais de um arquivo é usado o atributo 'multiple'

    Código: <label>Arquivo:</label> <input type="file" name="arquivo" multiple>


  • search: Campo de pesquisa, similar ao text, mas otimizado para buscas.
    • Código: <label>Pesquisar:</label><input type="search" name="busca" placeholder="Pesquisar...">


  • hidden: Campo invisível, usado para enviar dados ocultos no formulário.
    • Código: <input type="hidden" name="id_usuario" value="123">
    (não aparece na tela, mas é enviado no formulário)

  • submit: Botão para enviar o formulário.
    • Código: <input type="submit" value="Enviar">


  • reset: Botão para limpar os campos do formulário.
    • Código: <input type="reset" value="Limpar">

Note que o input é uma tag autocontida, não percisa ser aberta e depois fechada. Já a tag label sempre precisa ser aberta e fechada. Existem mais tipos de inputs. É importante destacar também que alguns tipos de inputs utilizam mais alguns atributos da tag como 'name', 'value', 'min', 'max'... Esses atributos são especificos de cada tipo, qualquer dúvida visite w3schools - Inputs

Textarea

O elemento <textarea> em HTML cria um campo de texto multilinha, ou seja, um espaço onde o usuário pode digitar várias linhas de texto. Com o atributo rows desterminando o númerode linhas e o atributo cols determinando o número de coluns

Código:  <label>Mensagem:</label><textarea id="mensagem" name="mensagem" rows="5" cols="30">

Select e Option

O elemento <select> em HTML cria uma lista suspensa (menu dropdown) que permite ao usuário escolher uma ou mais opções entre várias pré-definidas. Ela é usada juntamente com a tag <option> que nos fornecerá as opções que podem ser escolhidas.

Código:
<label>Escolha um curso:</label>
<select id="curso" name="curso">
<option value="html">Curso de HTML</option>
<option value="css">Curso de CSS</option>
<option value="javascript">Curso de JavaScript</option>
</select>

No exemplo acima é possível selecionar apenas 1 dos cursos. Porém se usarmos o atributo 'multiple' dentro da tag <select> podemos selecionar mais de 1 curso.

Button

O elemento <button> em HTML cria um botão clicável que pode executar diferentes ações dentro de um formulário ou em scripts. Seu funcionamento é parecido com o da tag <input type="submit"> ou <input type="reset">, porém oferece mais flexibilidade, pois permite incluir texto, imagens ou outros elementos dentro do botão.

Código: <button type="submit">Enviar</button>
        


Código: <button type="reset">Limpar</button>
        


Código: <button type="button" onclick="alert('Olá!')">Clique aqui</button>

Fieldset

Por fim temos o <fieldset>. Ele é usado para agrupar elementos de um formulário, criando uma caixa visual ao redor deles. Normalmente, é utilizada junto com a tag <legend>, que adiciona um título à caixa.

Código:
<form method="GET">
    <fieldset>
    <legend>Informações Pessoais</legend>
    <label>Nome:</label> <input type="text" name="nome">
    <label>Idade:</label> <input type="number" name="idade">
    <br><br>
    <button type="submit">Enviar</button>
    <button type="reset">Limpar</button>
    </fieldset>
</form>
Informações Pessoais

Esse é um exemplo funcional de todos os conceitos vistos sobre formulários. Aqui foram utilizados apenas 2 tipos de inputs (text e number) e 2 tipos de buttons (submit e reset). E no formulário foi aplicada o atributo method="GET" que envia os dados do formulário pela URL, tornando-os visíveis e fáceis de compartilhar. Se preencher os campos e clicar em enviar, a URL da página será atualizada mostrando os valores de nome e idade inseridos. E caso clique em limpar todas a informações do formulário serão apagada.

DIV e SPAN

O que é uma Div?

A tag <div> no HTML é usada para criar divisões ou blocos genéricos na página. Ela não tem significado semântico, servindo apenas para agrupar outros elementos e facilitar a organização, estilização (com CSS) ou manipulação (com JavaScript) do conteúdo. Todo o conteúdo dessa página está sepado em div's. Ela é do tipo block-level (bloco). Isso significa que, por padrão, ela ocupa toda a largura disponível e sempre começa em uma nova linha, separando visualmente o conteúdo dos outros elementos. Na pratica ao se declarar 2 divs a primeira ficará por cima da segunda. É possível mudar essa caracteristica através do CSS, definindo as div's como inline-block (bloco em linha) permitindo que os elemetos fiquem na mesma linha. Abaixo está um trecho de código em CSS configurando algumas div's, esse trecho de código se encontra dentro do head no nosso HTML.

<style>
  .bloco {
      width: 200px;
      height: 100px;
      margin: 10px;
      padding: 10px;
      color: #fff;
      font-weight: bold;
      display: inline-block;
      text-align: center;
      vertical-align: top;
    }
    .vermelho { background: #e74c3c; }
    .azul { background: #3498db; }
    .verde { background: #2ecc71; }
</style>

Feito a configuração no CSS, agora é possível implementar as div's.

Código: <div class="bloco vermelho">Bloco Vermelho</div>
        <div class="bloco azul">Bloco Azul</div>
        <div class="bloco verde">Bloco Verde</div>
Bloco Vermelho
Bloco Azul
Bloco Verde

No exemplo acima foi utilizado o atributo class = "bloco ...", esse atributo serve para agrupar elmentos que serão estilizados da mesma forma no CSS. Em suma, as <div> servem para agrupar e organizar elementos em blocos na página HTML. Elas são do tipo block-level, ocupam toda a largura disponível e facilitam a estruturação do layout, aplicação de estilos com CSS e manipulação com JavaScript. São essenciais para criar seções, áreas e dividir o conteúdo visualmente em um site.

O que é um Span?

A tag <span> é um elemento HTML do tipo inline, usada para marcar ou agrupar pequenas partes de texto ou outros elementos dentro de uma linha. Ela não tem significado semântico, servindo apenas para aplicar estilos com CSS ou manipular com JavaScript. 
Código CSS: 
.destaque {
    color: blue;
    font-weight: bold;
}   
Código HTML: 
<p>Este texto tem uma <span class="destaque">palavra destacada</span> usando span.</p>

Este texto tem uma palavra destacada usando span.

Diferente da <div>, que é um elemento de bloco (block-level) e ocupa toda a largura disponível, o <span> é um elemento inline. Isso significa que o <span> não quebra a linha e só ocupa o espaço necessário para o seu conteúdo, sendo usado para destacar ou estilizar partes pequenas de texto dentro de uma linha, sem criar um novo bloco visual.

Mídias

Imagem

A tag img é usada para incorporar imagens em uma página HTML. Ela é um elemento vazio, o que significa que não possui uma tag de fechamento. 

    O atributo src é obrigatório e especifica o caminho para o arquivo de imagem. 

    O atributo alt é usado para fornecer um texto alternativo que descreve a imagem, útil para acessibilidade e quando a imagem não pode ser carregada.

    O atributo width define a largura da imagem em pixels.

    O atributo height define a altura da imagem em pixels.

    O atributo title fornece informações adicionais sobre a imagem, que geralmente aparecem como uma dica de ferramenta quando o usuário passa o mouse sobre a imagem.
Aqui seguem 2 exemplos de imagens, a primeira armazenada localmente e a segunda proveniente do site da w3schools. 

Exemplo 1:
 <img src="ballons.png" alt="3 balões coloridos: 1 azul, 1 vermelho e 1 amarelo" title="Balões Coloridos" width="300" height="200">
    
Exemplo 2:
<img src="https://www.w3schools.com/html/pic_trulli.jpg" alt="Trulli" title="Pic Trulli" width="500" height="333">

Áudio

Ao trabalhar com áudio em HTML, utilize a tag <audio>.A tag audio é usada para incorporar conteúdo de áudio em uma página HTML. Ela permite que você adicione arquivos de áudio, como música ou narração, que os usuários podem reproduzir diretamente no navegador. Ela é usada juntamente com alguns atributos para refinar seu uso, sendo alguns deles:

    O atributo controls adiciona controles de reprodução, como play, pause e volume, para que os usuários possam interagir com o áudio.

    O atributo autoplay faz com que o áudio comece a tocar automaticamente quando a página é carregada. Deve ser usado com cautela, pois pode ser intrusivo para os usuários.

    O atributo loop faz com que o áudio seja reproduzido em loop contínuo, reiniciando automaticamente quando chega ao fim.

    O atributo muted inicia o áudio sem som. Pode ser útil em conjunto com autoplay, já que muitos navegadores bloqueiam a reprodução automática de áudio com som.

    O atributo src especifica o caminho para o arquivo de áudio. Pode ser usado diretamente na tag audio ou dentro de uma tag source para suportar múltiplos formatos de áudio. 
    É comum usar multiplas tags source para garantir compatibilidade com diferentes navegadores.

Exemplo de audio retirado diretamente da internet:

Código: 
<audio controls> 
    <source src="https://www.w3schools.com/html/horse.ogg" type="audio/ogg">
    <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg">
        Seu navegador não suporta o elemento de áudio.
</audio>

    O atributo controls adiciona controles de reprodução, como play, pause e volume, para que os usuários possam interagir com o vídeo.

    O atributo autoplay faz com que o vídeo comece a tocar automaticamente quando a página é carregada. Deve ser usado com cautela, pois pode ser intrusivo para os usuários.

    O atributo muted inicia o vídeo sem som. Pode ser útil em conjunto com autoplay, já que muitos navegadores bloqueiam a reprodução automática de vídeos com som.

Track

A tag track é usada para adicionar faixas de texto, como legendas, descrições ou capítulos, a elementos de mídia como vídeo e áudio em uma página HTML. Ela melhora a acessibilidade e a experiência do usuário, permitindo que o conteúdo seja compreendido por um público mais amplo. Ela conta com alguns atributos como: 

    O atributo kind especifica o tipo de faixa que está sendo adicionada. Os valores comuns incluem

    caption -> usado para legendas que incluem não apenas o diálogo, mas também descrições de sons importantes para a compreensão do conteúdo.

    subtitles-> usado para legendas que traduzem o diálogo falado em outro idioma.

    descriptions-> usado para fornecer descrições de áudio para pessoas com deficiência visual, descrevendo o que está acontecendo na tela.

    chapters-> usado para dividir o conteúdo em capítulos, permitindo que os usuários naveguem facilmente entre diferentes seções do vídeo ou áudio.

    metadata-> usado para fornecer informações adicionais sobre o conteúdo, como créditos ou informações técnicas, que não são exibidas diretamente ao usuário.

    O atributo src define o caminho para o arquivo de texto que contém as legendas ou descrições. Esse arquivo geralmente está no formato WebVTT (.vtt).

    O atributo srclang especifica o idioma da faixa de texto usando um código de idioma (por exemplo, "en" para inglês, "pt" para português).

    O atributo label fornece um rótulo descritivo para a faixa, que pode ser exibido em menus de seleção de faixas no player de mídia.

    O atributo default indica que esta faixa deve ser ativada por padrão quando o vídeo ou áudio for carregado. Apenas uma faixa pode ter este atributo.

    É POSSÍVEL TER VÁRIAS TAGS TRACK PARA UM MESMO VIDEO OU AUDIO, CADA UMA COM UMA FUNÇÃO DIFERENTE.

Aqui seguem 2 exemplos em vídeo. O primeiro não está sendo utilizado o track. O segundo já está utilizando o track.

Código:
<video width="500" height="300" srclang="en">
    <source src="Guns N' Roses - Welcome To The Jungle.mp4" type="video/mp4">
    Seu navegador não suporta o elemento de vídeo.
</video>

(Sem Track)


Código:
<video width="500" height="300" srclang="en">
    <source src="Guns N' Roses - Welcome To The Jungle.mp4" type="video/mp4">
    <track kind="captions" src="Guns-N'-Roses-Welcome-To-The-Jungle.vtt" srclang="en" label="English" default>
    Seu navegador não suporta o elemento de vídeo.
</video>

Iframe

Por fim temos o Iframe: Essa tag é usada para incorporar outro documento HTML dentro do documento atual. Ela cria uma janela embutida que pode exibir uma página web diferente, permitindo a integração de conteúdo externo, como vídeos, mapas, formulários ou outras páginas web, diretamente na sua página. E conta com alguns atribustos como:

    O atributo src define o URL da página a ser incorporada.

    O atributo width define a largura do iframe em pixels ou como uma porcentagem do elemento pai.

    O atributo height define a altura do iframe em pixels ou como uma porcentagem do elemento pai.

    O atributo frameborder especifica se o iframe deve ter uma borda. Os valores possíveis são "0" (sem borda) e "1" (com borda). Este atributo está obsoleto em HTML5, e é recomendado usar CSS
    para estilizar a borda.

    O atributo allowfullscreen permite que o iframe seja exibido em tela cheia, se o conteúdo incorporado suportar essa funcionalidade.

    O atributo loading pode ser usado para definir o comportamento de carregamento do iframe. O valor "lazy" faz com que o iframe seja carregado apenas quando estiver prestes a entrar na área
    visível da tela, melhorando o desempenho da página.

    O atributo name atribui um nome ao iframe, que pode ser usado como destino para links ou scripts.

    O atributo sandbox aplica restrições adicionais ao conteúdo do iframe, como desabilitar scripts ou impedir a navegação para outras páginas. Pode ser usado com vários valores para definir
    diferentes níveis de restrição.

    O atributo referrerpolicy controla como as informações de referência são enviadas ao carregar o conteúdo do iframe. Pode ser usado para melhorar a privacidade e segurança.

Tabelas

As tabelas em HTML servem para organizar e apresentar dados em formato tabular, ou seja, em linhas e colunas. Elas são ideais para mostrar informações estruturadas, como planilhas, horários, notas, listas de produtos, relatórios, etc. A tag <table> é usada para criar uma tabela.

As tabelas em HTML devem ser utilizadas exclusivamente para apresentar dados tabulares, e não como ferramenta de layout ou design da página, pois existem técnicas mais adequadas como o uso de CSS para isso.

É fundamental incluir cabeçalhos de tabela (<th>) para identificar o conteúdo das colunas e linhas, o que melhora significativamente a acessibilidade e compreensão da tabela por todos os usuários.

Quando for necessário unir células, é possível utilizar os atributos colspan e rowspan, que permitem mesclar colunas ou linhas.

Além disso, é importante garantir a acessibilidade para pessoas que utilizam leitores de tela ou outros dispositivos assistivos, assegurando que os dados possam ser interpretados corretamente.

Também é recomendado evitar tabelas aninhadas, pois elas tornam a estrutura mais complexa e dificultam tanto a manutenção do código quanto a leitura do conteúdo.

A estilização das tabelas deve ser feita com CSS, evitando atributos HTML obsoletos, e o desenvolvedor deve testar as tabelas em diferentes dispositivos e navegadores para garantir que sejam exibidas corretamente em todos os contextos.

É aconselhável adotar tabelas responsivas, para que se ajustem bem a telas menores e proporcionem uma boa experiência de uso em dispositivos móveis.

Por fim, as tabelas contam com uma série de atribustos, sendo os principais: 

    - border: Define a borda da tabela.
    - cellpadding: Define o espaçamento interno das células.
    - cellspacing: Define o espaçamento entre as células.
    - width: Define a largura da tabela.
    - height: Define a altura da tabela.
    - align: Alinha a tabela (esquerda, direita, centro).
    - bgcolor: Define a cor de fundo da tabela.
    - summary: Fornece uma descrição da tabela para leitores de tela.
    - caption: Adiciona uma legenda à tabela.
    - colgroup: Agrupa colunas para aplicar estilos ou atributos.
    - col: Define atributos para colunas específicas.
    - align (em <td> e <th>): Alinha o conteúdo da célula (esquerda, direita, centro).
    - valign (em <td> e <th>): Alinha verticalmente o conteúdo da célula (topo, meio, base).
    - alingn = "center": centraliza a tabela na página

Conceitos Básicos: tr, th, td

Para estruturar as tabelas são utilizados 3 tags:

  • A tag <tr> define uma linha na tabela. (Linhas)
  • A tag <td> define uma célula de dados em uma linha. (Colunas)
  • A tag <th>define uma célula de cabeçalho em uma linha. (Cabeçalho)

Conceitos de thead, tbody e tfoot

Esixtem também 3 conceitos muito importante ao trabalhar com tabelas.

<thead> - É a seção de cabeçalho da tabela. Normalmente contém os títulos das colunas (geralmente com <th>). Fica sempre no topo da tabela e ajuda os leitores de tela e navegadores a entenderem a estrutura dos dados.

<tbody> - É o corpo principal da tabela, onde ficam as linhas de dados reais (com <tr< e >td>). Pode ter várias linhas e normalmente é a parte mais extensa da tabela.

<tfoot> - É o rodapé da tabela, usado para informações de resumo ou totais. Mesmo que visualmente apareça no fim, alguns navegadores processam o <tfoot> antes do <tbody> para otimizar a renderização.

Com eles já é possivel montar uma tabela melhor estruturada, da seguinte forma:

Código:

    <thead>
        <tr>
            <th colspan="2">Dados</th>
        </tr>
        <tr>
            <th title="Nome Completo">Nome</th>
            <th title="Idade">Idade</th>

Mais alguns conceitos e estilização com CSS

Por fim, ao se trabalhar com tabelas, é importante destacar o uso das CSS's. Com elas conseguimos uma tabela visualmente mais atrativa e elegantes. Com CSS, é possível personalizar cores, bordas, espaçamento, fontes, efeitos ao passar o mouse, entre outros detalhes. Deixando sua tabela mais organizada, com cores alternadas nas linhas, cabeçalho destacado e bordas suaves. Abaixo segue um trecho de código em CSS e uma tabela no mesmo modelo da última porém com o uso da CSS's.

Código em CSS:
table.estilizada {
    -webkit-box-shadow: 12px 14px 6px 5px rgba(0,0,0,0.24);
    box-shadow: 12px 14px 6px 5px rgba(0,0,0,0.24);
    border: solid 1px gray;
}

table.estilizada thead th {
    background-color: black;
    color: white;
}

table.estilizada tfoot td {
    background-color: lightgray;
}

table.estilizada tbody tr:nth-child(even) td {
    background-color: #f1f1f1; /* linhas pares */
}

table.estilizada tbody tr:hover td {
    background-color:  #ff0000;
    cursor: pointer; /* mouse vira mãozinha */
}
table.estilizada tr:nth-child(even) {
    background-color: burlywood;
}
table.estilizada tr:hover td {
    background-color: darkgray;
}
